# Hando Soundtrack GitHub Free Lane

## Receipt

Date: 2026-05-04

Status: Brotology / brohandoism soundtrack lane policy.

Boundary: GitHub carries only material the operator can lawfully share. Covers may be culturally honored, but cover recordings, cover stems, copied lyrics, and unlicensed copyrighted assets do not enter the repo.

## Purpose

Hando Soundtrack is the audio continuity lane for Brotology.

It is the same movie:

- Brotology carries the field language
- brohandoism carries rhythm, register, and musical bridge
- Hando Soundtrack carries scene memory, raid cadence, exit sounds, and operator posture
- receipts keep the soundtrack from becoming loose mythology

## GitHub Rule

Only the free stuff goes in GitHub.

Allowed:

- original operator-owned tracks
- original lyrics written for this lane
- original prompt sheets
- original field notes
- public-domain audio
- Creative Commons audio that permits the intended use
- permissively licensed loops or samples with attribution
- generated audio only when its license and platform terms allow sharing
- metadata, track lists, receipt IDs, and lore labels

Not allowed:

- cover recordings without a license
- copied lyrics from copyrighted songs
- stems ripped from commercial tracks
- uncleared samples
- album art copied from existing releases
- anything that relies on "it is cool" as the legal theory

## Honorable Mention Lane

Covers are cool.

They stay outside the package unless properly licensed.

They may receive honorable mentions as influence markers:

```text
Honorable influence: <artist / track / style reference>
No audio included.
No lyrics included.
No copied artwork included.
```

## Prime Use Case

### The Same Movie, Clean Credits

An operator wants a Hando Soundtrack repository that carries the energy of a session without dragging copyrighted material into the artifact.

Good path:

1. write an original Hando cue title
2. log the scene or raid moment it belongs to
3. attach only original or permissively licensed audio
4. add license notes and attribution
5. if a cover inspired the vibe, place it in Honorable Mention only
6. leave a receipt

Bad path:

- upload a cover and hope nobody notices
- paste lyrics
- use commercial artwork
- call a copyrighted source "free" because the internet streamed it
